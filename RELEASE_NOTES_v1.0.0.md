# Scripts Formula v1.0.0 - Instalador Windows com 1 clique

Primeira release oficial do Scripts Formula com foco em instalacao simples para usuarios leigos.

## Novidades

- Instalador Windows gerado com Inno Setup.
- Nome do programa instalado: Scripts Formula.
- Criacao de atalhos no Menu Iniciar e Area de Trabalho.
- Pos-instalacao automatica para preparar dependencias e configurar funcionalidades.
- Programa principal mantido via launcher Scripts.bat.
- Workflow de GitHub Actions para build e anexo automatico do instalador em Releases.
- README atualizado com link de download direto do instalador.

## Arquivo desta release

- Scripts-Formula-Setup.exe

## Observacoes

- Requer Windows e ambiente TQS.
- Se Python nao estiver instalado, o instalador orienta e abre a pagina oficial para download.
